# Notion Plotly

## Usage

Pass following parameters to URL.

- `type=?`: type of plot
  - `bar`: bar chart
- `x=1,2,3,4`: x values
- `y=1,2,3,4`: y values
- `group=A,A,B,B`: group values
- `sortBy=?`: sort values by `x` or `y` or `group`
